# Проект по автоматизации ремонта и обслуживания оборудования

Данный проект посвящён анализу и проектированию корпоративной информационной системы для автоматизации процессов технического обслуживания и ремонта (ТОиР) производственного оборудования. В ходе работы рассмотрены системы класса EAM/CMMS, выполнен сравнительный анализ, смоделирован бизнес-сценарий и разработано техническое задание на создание системы «Ремонт-Профи».
